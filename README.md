# ROKI BIOS
AC Power Loss off 桌機關，on Server開

設定AHCI or RST...也可

無法喚醒可能原因Fast Boot，選「Boot Options」→移到「Fast Boot」→改成 [ Disabled ]

Acer電腦，關閉Deep Power Off Mode (喚醒用)，選「Power」→移到「Deep Power Off Mode」→改成 [ Disabled ]
